# OneDegreeAdvisory Website

Premium static website for OneDegreeAdvisory, built from the supplied PDF direction.

## Files

- `index.html` - page structure, SEO metadata, content, and schema
- `styles.css` - responsive visual system, layout, and motion
- `script.js` - navigation, reveal animation, filters, carousel, finder, and form state

## Run Locally

Open `index.html` directly in a browser, or serve the folder with:

```bash
npx serve .
```
